# 🚗 ParkALot

ParkALot is an iOS application built with **Swift** and **Firebase** that helps users explore and manage parking information in Istanbul. Users can browse parking lots by district, view detailed availability information live from IBB API, and explore locations on an interactive map.

---

## ✨ Features

- 🔐 User authentication (Register / Login / Logout)
- 🏙️ Browse parking lots in Istanbul
- 📍 District-based categorization of parking areas
- 🅿️ Detailed parking lot information:
  - Open hours
  - Space availability
  - Location details
- 🗺️ Interactive map view with parking lot markers
- ☁️ Firebase integration for backend services

---

## 🛠️ Tech Stack

- **Language:** Swift  
- **Framework:** UIKit  
- **Backend:** Firebase (Auth + Firestore)  
- **Map Integration:** MapKit
- **IDE:** Xcode
